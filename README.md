# kamil_case_study
uPayment Case study


## Installation

git clone https://github.com/TheKamil-dev/kamil_case_study.git

Update Environment variables  in .env file.

Create Databse in mysql and update DB_DATABASE 

Run following command.
```
composer install

php artisan migrate

php artisan db:seed

Default user added
user: k@yopmail.com
password: case@study

```
